# nrfxlib APIs

## Introduction

The nrfxlib library provides a set of APIs from Nordic Semiconductor for
developing applications on nRF Series wireless connectivity devices.

It offers a collection of low-level and hardware-accelerated libraries
designed to optimize performance and simplify the development of
applications involving wireless communication and connectivity.

The library is structured as a set of independent components, each focusing
on specific functionality such as cryptography, networking, protocol stacks,
and hardware abstraction.

---

## Main API categories

### Cryptocell

Utilize hardware-accelerated cryptographic functionality to implement
enhanced security features.

### Nordic Security Module

Implement comprehensive security mechanisms including cryptographic
operations, secure storage, and key management.

### SoftDevice Controller

Leverage the SoftDevice Controller for Bluetooth Low Energy and ANT protocol
stacks, enabling efficient wireless communication.

### Modem library

Develop cellular IoT applications using the LTE modem library for
communication with cellular networks.

### ZBOSS Zigbee

Integrate the Zigbee protocol stack to enable reliable and secure mesh
networking for smart home and industrial applications.

### MPSL (Multiprotocol Service Layer)

Manage coexistence between multiple wireless protocols operating on the
same device.

### IEEE 802.15.4 radio driver

Implement the IEEE 802.15.4 radio driver for low-rate wireless personal
area networks.

### GNSS interface

Access Global Navigation Satellite System functionality for location-based
services and applications.

### Bluetooth Mesh

Develop applications using the Bluetooth Mesh networking standard for
large-scale device networks.

### Thread and Zigbee

Utilize Thread and Zigbee networking protocols to build reliable, scalable,
and secure IoT networks.

### Device Firmware Update (DFU)

Implement secure and reliable mechanisms for device firmware updates.
